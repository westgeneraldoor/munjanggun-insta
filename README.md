# 🚪 문장군 인스타그램 콘텐츠 운영 OS (v3.0)

본 프로젝트는 문장군(도어/중문/현관문 시공 전문 브랜드)의 인스타그램 계정 활성화와 발견 알고리즘을 통한 **DM 문의 및 무료 방문 실측 예약 전환**을 관리하는 독립형 콘텐츠 운영 시스템입니다.

---

## 🎯 핵심 운영 목표
* **발견 & 감정 엔진**: 단순 시공 이미지 나열이 아닌, 고객의 페인포인트(방문 썩음, 우풍, 좁은 현관 등)를 자극하여 공감대 형성
* **저장 & 공유 유도**: 인스타그램 알고리즘이 선호하는 **정보성 비교표, 체크리스트 캐러셀** 위주 발행으로 저장률 극대화
* **전환 최적화 (CTA)**: "무료 방문 실측(0원)", "본사 전속 시공팀 직접 운영", "추가금 제로"의 강점을 활용하여 프로필 링크 클릭 및 DM 예약으로 연결

---

## 🛠️ 기술 스택 및 제작 표준 (v3.0)

인스타그램 캐러셀 제작 시 복잡한 Node.js 이미지 빌드 환경(Puppeteer 등)의 의존성을 제거하고, **브라우저 프리뷰와 캡션 복사가 즉시 가능한 단일 HTML 표준**으로 전환하였습니다.

* **HTML5 / CSS3 (Vanilla)**: 디자인 가이드에 정의된 컬러 토큰 및 타이포그래피 상속
* **Montserrat + Noto Sans KR**: 세련되고 현대적인 인테리어 톤앤매너 폰트 구성
* **CSS Variables (Mocha v2.3 테마)**:
  * `--accent`: `#8B7355` (모카 브라운 강조색)
  * `--cream`: `#F5F0E8` / `--cream2`: `#EDE8DD` (아늑하고 세련된 인테리어 배경색)
  * `--dk-cover`: `#1E1C18` (다크 웜 커버색)

## 🚀 v3.0 HTML 캐러셀 사용 및 발행 방법

### 1단계: 템플릿 복제 및 편집
1. `templates/CAROUSEL_HTML_TEMPLATE.html`을 복사하여 `instagram/content/NNN_테마명.html` 형식으로 저장합니다.
2. 기획 내용에 맞춰 제목, 슬라이드 본문, 이미지 캡션 설명을 작성합니다.

### 2단계: 브라우저 프리뷰 검수
* 작성 중이거나 작성이 완료된 HTML 파일을 **더블클릭**하여 웹 브라우저(크롬, 엣지 등)에서 즉시 실제 모바일 1080x1350(또는 축소 540x675) 해상도로 렌더링된 캐러셀을 비주얼 검수합니다.

### 3단계: 이미지 실사진 매핑
* `[AppSheet 현장 실사진 권장...]`으로 표기된 이미지 슬롯 영역(`.image-slot`)에 어울리는 문장군 시공 실사진 경로를 지정합니다.
  ```html
  <div class="image-slot" style="background-image: url('실사진경로.jpg'); background-size: cover;">
  ```

### 4단계: 캡션 및 해시태그 원클릭 복사
1. HTML 파일 내 7번째 슬라이드인 **Caption Copy Card**로 이동합니다.
2. 브라우저 프리뷰 화면 우측 또는 하단의 **`📋 캡션 복사하기`** 버튼을 누르면 본문 문구와 해시태그가 한 번에 클립보드에 복사됩니다.
3. 인스타그램 게시글 업로드 시 해당 복사본을 그대로 붙여넣어 발행합니다.
