# 문장군 인스타그램 포스팅 레지스트리

> 인스타그램 콘텐츠의 발행 이력과 성과를 추적하는 등록부.
> 블로그 POSTING_REGISTRY.md와 동일한 역할.
> 최종 업데이트: 2026-05-21

---

## 사용법

1. **제작 전:** 기발행 목록 확인 → 중복 방지, 성과 좋았던 패턴 참고
2. **제작 후:** 아래 테이블에 새 콘텐츠 추가 (파일 경로 기록)
3. **발행 후:** URL, 발행일, 성과 데이터 업데이트
4. **1주 후:** 비율 컬럼 (저장률, DM전환율) 계산 업데이트

### 비율 계산법

| 비율 | 공식 | 의미 |
|------|------|------|
| 저장률 | 저장 ÷ 도달 × 100 | 콘텐츠 정보 가치 (높을수록 강한 콘텐츠) |
| DM전환율 | DM문의 ÷ 프로필방문 × 100 | 전환 효율 (높을수록 CTA 효과적) |

## 성과 패턴 메모

> 발행 후 성과 데이터가 쌓이면 아래에 패턴을 기록한다.
> 다음 콘텐츠 제작 시 작업자가 자동 참조.

### 해석 가이드

| 패턴 | 의미 | 대응 |
|------|------|------|
| 도달 높음 + 저장률 낮음 | 훅은 강하지만 정보 가치 부족 | 체크리스트/비교표 보강 |
| 도달 낮음 + 저장률 높음 | 소수에게 강렬, 알고리즘 미노출 | 커버 훅 강화 or 광고 부스트 |
| 프로필방문 높음 + DM전환율 낮음 | 관심은 있으나 행동 장벽 존재 | CTA/프로필 링크 동선 점검 |
| DM문의 있음 + 예약전환 없음 | 응대 또는 예약 흐름 문제 | DM 스크립트/네이버예약 동선 점검 |
| 공유 높음 | 상황 공감형 콘텐츠 강세 | 유사 테마 변주 확대 |
