# 문장군 인스타그램 콘텐츠 스코어카드

> 목적: 제작된 콘텐츠의 품질을 8차원으로 평가하여 일관된 기준을 유지한다.
> AI 자동 검수, A/B 비교, 성과 패턴 학습의 기반 문서.
> 최종 업데이트: 2026-05-21

---

## 평가 체계

**총점: 80점 만점** (8차원 × 10점)

| 등급 | 점수 | 판정 |
|------|------|------|
| S | 72~80 | 즉시 발행 |
| A | 64~71 | 경미한 수정 후 발행 |
| B | 56~63 | 주요 항목 보완 필요 |
| C | 48~55 | 구조 재설계 필요 |
| F | 47 이하 | 폐기 후 재제작 |

**최소 기준: B등급(56점) 이상이어야 발행 가능**

## 8차원 평가 기준

### 1. HOOK POWER (훅 파워) — 10점

> 3초 안에 스크롤을 멈추게 하는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 숫자 훅 + 감정 자극 + 시각적 대비가 동시에 작동 |
| 7~8 | 숫자 또는 경고형 훅이 명확하고, 커버 디자인이 강렬함 |
| 5~6 | 훅이 있지만 평범하거나 시각적 임팩트가 약함 |
| 3~4 | 훅 없이 정보 나열로 시작 |
| 1~2 | "안녕하세요" 도입부 또는 브랜드명 먼저 노출 |

채점 포인트:
- 커버 텍스트가 7단어 이내인가?
- 숫자, 질문, 경고 중 하나 이상 사용했는가?
- 커버 이미지/배경이 텍스트를 방해하지 않는가?

---

### 2. SAVEABILITY (저장 유도력) — 10점

> "나중에 시공할 때 다시 봐야지" 반응을 일으키는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 체크리스트/비교표/단계별 가이드가 있어 저장 안 하면 손해 |
| 7~8 | 정보 밀도가 높고 구조화되어 있어 한 번에 소화 어려움 |
| 5~6 | 유용한 정보가 있지만 저장까지 유도하는 구조는 약함 |
| 3~4 | 일반적인 정보 제공 수준 |
| 1~2 | 저장 가치가 없는 단순 홍보 |

채점 포인트:
- 체크리스트, 비교표, 단계별 가이드 중 하나 이상 있는가?
- "📌 저장" CTA가 자연스럽게 삽입되어 있는가?
- 정보가 구조화되어 있는가? (번호, 아이콘, 표 등)

---

### 3. SHAREABILITY (공유 유도력) — 10점

> "이거 우리 집도 해당되는데?" 태그 반응을 일으키는가?

| 점수 | 기준 |
|------|------|
| 9~10 | "@친구 이거 봐" 반응이 자연스럽게 나오는 상황 공감형 |
| 7~8 | 특정 상황(신혼, 아이, 반려동물)에 해당하는 사람이 공유할 만함 |
| 5~6 | 유용하지만 공유보다는 개인 저장에 가까움 |
| 3~4 | 공유 욕구를 자극하는 요소 없음 |
| 1~2 | 광고성이 강해 공유하면 민폐 |

채점 포인트:
- 고객 대화체 인용이 있는가?
- 특정 상황/페르소나를 겨냥하는가?
- Before/After 극적 대비가 있는가?

---

### 4. DM INTENT (DM 전환 의도) — 10점

> "나도 문의해볼까?" 행동으로 이어지는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 구체적 상황 제시 + 무료 실측 CTA + 진입 장벽 제거가 완벽 |
| 7~8 | 무료 실측 CTA가 자연스럽고, 비용 부담 없음을 명시 |
| 5~6 | CTA가 있지만 "왜 문의해야 하는지" 설득이 약함 |
| 3~4 | CTA가 형식적이거나 광고 느낌 |
| 1~2 | CTA 없음 또는 "지금 바로 구매하세요" 수준 |

채점 포인트:
- "프로필 링크" CTA가 자연스럽게 삽입되었는가?
- "무료", "0원", "부담 없이" 등 진입 장벽 제거 표현이 있는가?
- 고객이 자기 상황과 연결할 수 있는 맥락이 있는가?

---

### 5. TRUST (신뢰 형성) — 10점

> "이 업체는 진짜 잘하는구나" 인식을 심는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 실제 현장 사진/영상 + 전문 지식 + 리뷰 근거가 자연스럽게 조합 |
| 7~8 | 전문가 톤으로 실질적 정보를 제공하며 신뢰감 있음 |
| 5~6 | 정보는 있지만 "문장군이어야 하는 이유"가 약함 |
| 3~4 | 일반적인 정보 제공, 업체 차별점 없음 |
| 1~2 | AI 느낌이 강하거나 근거 없는 주장 |

채점 포인트:
- 시공 결과물에 실제 사진을 사용했는가? (AI 이미지 금지)
- 문장군 강점(무료 실측, 전속 시공팀, 리뷰)이 자연스럽게 녹았는가?
- 전문 용어를 쉽게 풀어서 설명하는가?

---

### 6. VISUAL IMPACT (시각적 임팩트) — 10점

> 피드에서 "이건 뭐지?" 시선을 끄는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 모카 브라운 톤 통일 + 상하 50:50 + 타이포 강조 완벽 |
| 7~8 | 디자인 시스템을 따르고 시각적 완성도 높음 |
| 5~6 | 디자인 시스템을 따르지만 밋밋하거나 여백이 아쉬움 |
| 3~4 | 디자인 가이드 미준수, 컬러/폰트 불일치 |
| 1~2 | 기본 템플릿 느낌, 디자인 없음 |

채점 포인트:
- CAROUSEL_DESIGN_GUIDE v3.0 컬러/폰트/레이아웃을 준수하는가?
- 상하 50:50 분할이 지켜지는가?
- TIP 박스, 구분선, 넘버링 등 구조적 요소가 있는가?

---

### 7. LOCAL SEO (지역 연결성) — 10점

> 서비스 지역 고객이 "우리 동네도 되나?" 반응하는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 서비스 지역 자연 노출 + 지역 해시태그 + 지역 맥락 콘텐츠 |
| 7~8 | 해시태그에 지역명 포함, 캡션에 서비스 지역 암시 |
| 5~6 | 해시태그에만 지역 포함 |
| 3~4 | 지역 관련 요소 없음 |
| 1~2 | 불가 지역을 포함하는 오류 |

채점 포인트:
- 해시태그 뱅크에서 지역 해시태그를 포함했는가?
- 불가 지역(영종도, 연천 등)을 가능하다고 표현하지 않았는가?
- 캡션에 서비스 지역이 자연스럽게 언급되는가?

---

### 8. BRAND FIT (브랜드 적합도) — 10점

> 문장군 톤과 브랜드 규칙을 지키는가?

| 점수 | 기준 |
|------|------|
| 9~10 | 톤(구어체 30%+), 금지어 0개, 제품명 정확, CTA 방향 완벽 |
| 7~8 | 대부분 준수, 사소한 톤 불일치 1~2건 |
| 5~6 | 브랜드 규칙은 지키지만 톤이 블로그체 또는 AI체 |
| 3~4 | 금지어 사용, 제품명 오류, CTA 방향 위반 중 1건 |
| 1~2 | 브랜드 규칙 다수 위반 |

채점 포인트:
- AI 냄새 표현(물론/또한/더불어/이처럼/결론적으로) 사용 여부
- 없는 제품 언급 여부
- 가격 직접 언급 여부
- "안녕하세요 문장군입니다" 도입부 사용 여부
- 구어체 비율 30% 이상 유지 여부

## 사용법

### AI 자동 채점

콘텐츠 제작 완료 후, 다음 형식으로 자체 채점한다:

```
## 스코어카드: 013_우리집중문선택가이드

| 차원 | 점수 | 근거 |
|------|------|------|
| HOOK POWER | 8 | 숫자 훅 "3가지" + 경고형 "모르면 후회" 조합 |
| SAVEABILITY | 9 | 3종 비교표 + 체크리스트 구조 |
| SHAREABILITY | 6 | 범용 주제라 특정 상황 공감은 약함 |
| DM INTENT | 7 | 무료 실측 CTA 자연스러움 |
| TRUST | 8 | 전문 용어 풀이 + 실측 과정 설명 |
| VISUAL IMPACT | 8 | 모카 톤 통일, 비교표 시각적 임팩트 |
| LOCAL SEO | 7 | 지역 해시태그 포함, 캡션에 서비스 지역 암시 |
| BRAND FIT | 9 | 톤 완벽, 금지어 0, 구어체 30%+ |
| **합계** | **62/80** | **A등급 — 경미한 수정 후 발행** |
```

### 성과 기반 보정

발행 후 실제 성과 데이터와 채점을 비교한다:
- 채점 높음 + 성과 낮음 → 채점 기준 재검토
- 채점 낮음 + 성과 높음 → 놓친 성공 요인 분석
- 10편 누적 시 채점 가중치 조정
