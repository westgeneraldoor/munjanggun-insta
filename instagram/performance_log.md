# Instagram Performance Log

> 인스타그램 콘텐츠별 성과를 기록하는 운영 로그다.
> 대시보드보다 이 파일을 먼저 채운다.

## 입력 원칙

- 업로드 후 24시간, 7일, 14일 기준으로 업데이트한다.
- 숫자를 모르면 빈칸으로 둔다. 추정치를 쓰지 않는다.
- 최소 입력 필드는 반드시 채운다: 게시일, 콘텐츠 ID, 포맷, 목적 태그, 저장수, DM 기여 여부.
