# Instagram Content Calendar

> 월간 인스타그램 발행 계획과 실제 발행 이력을 함께 관리한다.
> 캘린더는 대행사 브리프보다 상위 계획이며, 각 슬롯에서 필요한 경우 브리프를 생성한다.

## 캘린더 원칙

- 캘린더가 먼저 전체 슬롯을 잡는다.
- 각 슬롯은 목적 태그와 포맷을 가진다.
- 외부 제작이 필요한 슬롯만 `agency_brief.md`로 분리한다.
- 발행 후 실제 게시일과 성과 로그 ID를 연결한다.

## 브리프 연결 규칙

- `대행사 브리프 필요 = Y`인 슬롯만 `posts/.../instagram/agency_brief.md`를 만든다.
- 브리프에는 반드시 슬롯 ID를 적는다.
- 성과 리뷰는 브리프 단위가 아니라 캘린더 슬롯 ID 기준으로 기록한다.
